# Corporate Financial Forecasting & Valuation Engine

One-liner: Built a corporate forecasting + Monte Carlo valuation engine (Python), Streamlit demo, and Power BI dashboard.

## Quickstart (macOS)
1. `cd financial-forecasting-engine`
2. `source venv/bin/activate`  # activate the venv
3. `pip install -r requirements.txt`
4. Run notebook in VS Code or Jupyter:
   - `jupyter notebook` or open `notebooks/model_dev.ipynb` in VS Code
5. Run Streamlit when ready:
   - `streamlit run app.py`
